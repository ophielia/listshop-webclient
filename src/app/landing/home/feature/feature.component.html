features of the list shop

  • multiple lists

    Create multiple lists, and easily move or copy items between lists.

  • Filter

    Hide crossed items - or show only your crossed off items, as you like.

  • Offline mode

    Seamlessly transfers into offline mode when there isn't an internet connection.

  • Add by...

    Add to your list by item, by list, or by dish.

  • Personalized

    keeps the items which you add and remove from lists handy.

  • Get a jump start

    Create a starter list of the things you get - or want to remind yourself to get) each big shopping trip.
